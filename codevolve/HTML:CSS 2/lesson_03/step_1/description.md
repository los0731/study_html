# Business Card

게임 회사에 지원하려고 하니, 센스 있는 명함을 직접 만들어서 제출하라고 합니다. 
이런 명함은 어떠신가요? 이번 레슨에서는 온라인 명함을 만듭니다. 


## 이 페이지의 구조

이 페이지는 명함 안에 순서대로 이미지, 이름, 리스트가 있는 구조입니다. 그리고 리스트의 항목들로 직업, 전화번호, 이메일이 있습니다. 

```
|- 명함
    |- 이미지
    |- 이름
    |- 리스트
	    |- 직업
	    |- 전화번호
	    |- 이메일
```
