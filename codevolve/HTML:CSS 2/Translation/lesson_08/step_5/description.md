# CSS
## body
이 페이지는 전체적으로 `arial` 서체를 사용합니다. 하나씩 스타일을 적용해 갑시다.
* The margin of `body` is `0`.
* The font is `arial,sans-serif`.


**Instructions**
1. Apply style to `body`.
    ```css
    body {
      margin: 0;
      font-family: arial,sans-serif;
    }
    ```



## a Tag
역시 모든 `a`에 공통적으로 `underline`을 제거합니다. 
* `a`의 글씨의 장식이 없습니다.


**Instructions**
1. Apply style to `a`.
    ```css
    a {
      text-decoration: none;
    }
    ```
