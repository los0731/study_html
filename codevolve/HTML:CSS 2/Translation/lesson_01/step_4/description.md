# CSS
On this page the page title is `h1`, `h6` is posting, `h2` subheadline and `p` is the body. 



### Headline 

* Margin of `h1` is `40px 0 8px 0` for top/right/down/left.
* The font size is `20px`.


**Instructions**
1. Apply style to `h1`tag.
    ```css

      font-size: 20px;
    }
    ```
